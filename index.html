German-Flip: Memory Card Game with German Vocabulary

Learning German Vocabulary by playing

Whether you're a beginner or want to improve your language skills, this memory card game offers an engaging and enjoyable way to learn and practice German vocabulary.

Game rules:

  • The game’s idea is to match two of the same cards.
  • Match all 8 card pairs.
  • Start the game by clicking on any card to flip it over.
  • Each turn, you can only turn over 2 cards.
  • You complete the game after finding all 8 pairs.
  • Restart the game by clicking on “Reset Game” button.

Have fun!
